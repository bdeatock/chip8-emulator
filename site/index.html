Chip-8 Interpreter

Cycles per second (700Hz is a good value for most ROMs)

About

This is my Chip-8 interpreter, written in Go, brought to your browser through WebAssembly.

Chip-8 is a simple, old-school virtual machine from the 70s, originally used to run games on early computers. It's often mentioned as a great first project for those interested in building emulators.
